Bu proje, ASP.NET Core 8.0 kullanılarak geliştirilmiş, içinde kullanıcı kayıt, giriş, rol yönetimi ve e-posta doğrulama süreçlerini barındıran kapsamlı bir kimlik doğrulama (Identity) uygulamasıdır.

 Özellikler
ASP.NET Core Identity: Kullanıcı yönetimi, parola politikaları ve kilitlenme (lockout) mekanizmaları.


Entity Framework Core: SQL Server entegrasyonu ile veritabanı yönetimi.

E-posta Gönderimi: Kayıt sonrası e-posta onayı için SMTP entegrasyonu.

Rol Tabanlı Yetkilendirme: Özel poliçeler (Policy) ile esnek admin kontrolleri.

Seed Data: Başlangıçta test verilerinin otomatik oluşturulması.

Cookie Yönetimi: 15 günlük kalıcı oturum (Sliding Expiration) desteği.



 Kullanılan Teknolojiler


.NET 8.0 SDK 


Entity Framework Core 8.0.0 


SQL Server 

MVC Tasarım Deseni
